Flickr gallery integration for WordPress

== Description ==

A plugin to easily integrate flickr albums/photosets into your WordPress installation.

Features:
- Display single albums
- Display an overview of all the albums
- Fullscreen gallery mode
- Custom design
- Performance optimization

If you have any questions or feature requests, feel free to contact us via support@webkinder.ch.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Flickr Gallery screen to add new flickr users
4. Go to the Flickr Gallery Tab in the left menu bar and create a new gallery
5. Add the gallery to your page by using the shrotcode displayed on top of the specific gallery
